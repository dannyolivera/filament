## About

Filament application developed with the following technologies.

- Laravel [10]
- Laravel livewire [3]
- Tailwindcss [3]
- Filament [3]

## Run composer
``` bash
composer install
```
## Run migrations
``` bash
php artisan migrate
```
## Create user admin
``` bash
php artisan make:filament-user
```
## Run artisan
``` bash
php artisan serve
```
